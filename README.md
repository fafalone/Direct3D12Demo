# Direct3D12Demo
Using D3D12 with HLSL shaders in twinBASIC

<img width="841" height="503" alt="image" src="https://github.com/user-attachments/assets/ba2243a5-efd7-4c12-a529-36a0febbf264" />

This just draws a cube but it's a fairly deep intro dive into Direct3D 12.

**Requires Windows Development Library for twinBASIC v9.3.676 or newer**

DX12CubeDemo.twinproj - Uses custom window

DX12CubeDemoFrm.twinproj - Uses standard Form


**UPDATE**

I've added a second sample - DX12CubeDemoWithInput.twinproj. This demo the cube movement and rotation are controlled by user input with two options:

XInput: The cube is moved by the D-pad and rotated by the left stick.

Keyboard: The arrow keys move the cube, and W A S D rotates it.
